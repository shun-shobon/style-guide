# @shun-shobon/prettier-config

Shared Prettier config for shun-shobon's web projects.

## Installation

`prettier` is a peer dependency and must be installed in your project.

```bash
# If you use npm
npm install --save-dev @shun-shobon/prettier-config prettier

# If you use yarn
yarn add --dev @shun-shobon/prettier-config prettier

# If you use pnpm
pnpm add --save-dev @shun-shobon/prettier-config prettier
```

## Usage

```js
import { shun_shobon } from "@shun-shobon/prettier-config";

export default shun_shobon();
```
